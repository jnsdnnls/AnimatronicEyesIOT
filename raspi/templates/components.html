WHAT COMPONENTS WHERE USED?

Creating animatronic eyes involves integrating various electronic components, each serving a specific purpose to bring lifelike motion and functionality. Below is a detailed overview of the key components used in this project, along with their roles and importance. By understanding the function of each part, you can gain insight into the intricate design and engineering behind animatronic technology.

ESP32 WROOM-32

The ESP32 WROOM-32 is a powerful microcontroller with integrated Wi-Fi and Bluetooth capabilities. It's used as the main control unit for the animatronic eyes, enabling wireless communication and control. Its high processing power and versatility make it ideal for handling the complex operations required for animatronics.

9G Micro Servo

The 9G micro servo motors are essential for the movement of the animatronic eyes. These small yet powerful servos enable precise and smooth motion, ensuring the eyes can move accurately in various directions as programmed or controlled by the joystick.

PCA9685

The PCA9685 is a PWM (Pulse Width Modulation) driver that allows for the control of multiple servo motors simultaneously. It is used to manage the movements of the animatronic eyes and eyelids by providing precise control signals to the servos, enabling smooth and coordinated actions.

Raspberry PI

The raspberry PI is essential for handling the complex operations required for animatronics. It is used as the main network using RaspAP and MQTT to control the animatronic eyes. It is also used to host the web server that serves the web interface for controlling the animatronic eyes.
